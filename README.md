### 👨‍💻 À propos

Ce projet a été réalisé et structuré par moi-même pour faciliter le calcul des moyennes (Système sur 4.0 & Base 20) en fin de semestre. 

L'interface a été conçue en "Glassmorphism" pour être à la fois lisible sur ordinateur et sur mobile ! Elle inclut désormais un assistant IA (alimenté par le backend Python) pour prodiguer des conseils académiques basés sur les notes obtenues.
